# LF ENGENHARIA - CORE SaaS

## Business Goal
Sistema SaaS B2B "Enterprise Ready" focado em empreiteiras e construtoras, para acompanhamento gerencial, financeiro e operacional de obras. O objetivo é a diretoria possuir uma visão consolidada e em tempo real sobre alocação de recursos, desvios e relatórios (RDO).

## Aesthetic / Design
Padrão Silicon Valley. Estética Apple-like.
1. Foco em Dark Mode Profundo (#0A0A0A)
2. UI baseada no conceito de Bento Box Grid (Assimétrico, cantos muito arredondados)
3. Elementos Ultra-Premium (Glassmorphism, translúcidos, overlays sutis)
4. Layout fluido capaz de suportar Monitores Ultrawide 

## Core Modules
1. **Master Dashboard B2B:** Concentrador de Análises Financeiras e Obras
2. **Timeline de Acompanhamento (Feed RDO):** Módulo tipo rede-social para postagem de progresso visual de obras
3. **Task Manager:** Gestão de gargalos, pendências de engenheiros em obra
4. **CRM NGHUB OS / Base:** Tubo de clientes e funil (Conectado via Supabase)
